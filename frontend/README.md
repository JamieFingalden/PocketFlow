# 前端项目

## 项目结构

```
frontend/
├── mobile/           # 移动端应用
│   ├── ios/         # iOS原生应用
│   └── uniapp/      # Uniapp跨平台移动应用(Android)
└── web-admin/       # Web管理界面
```

## 技术栈

### 移动端
- iOS: Swift/SwiftUI原生开发
- Android: Uniapp跨平台框架
- Vue3 + TypeScript
- HBuilderX开发工具
- uni_modules组件库

### Web管理界面
- Vue3 + TypeScript
- Element Plus UI框架
- Vue Router
- Pinia状态管理
- Axios HTTP客户端
